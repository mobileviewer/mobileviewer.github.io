# Mobile Viewer

**Professional Responsive Website Tester & Device Preview Tool**

![Mobile Viewer](https://mobileviewer.github.io/favicon.png)

**See your website on any device — instantly.**

Mobile Viewer is the next-generation, browser-based tool trusted by developers, designers, and agencies to preview, test, and optimize responsive websites across real device viewports without installing any software or buying physical devices.

## Features

- Instant live preview of any website (including localhost/staging)
- 20+ real device presets (iPhone 15, Galaxy S23, iPad Pro, etc.)
- Portrait & Landscape orientation toggle
- Multi-device preview grid
- Responsive Viewport Resizer with draggable breakpoints
- Device Frame Simulator for client presentations
- Built-in QR Code Generator for instant mobile testing
- Website Screenshot Capturer
- Accessibility Contrast Checker (WCAG)
- User Agent Switcher
- Offline Mode Tester
- Page Speed & Core Web Vitals insights
- Breakpoint Tester Bookmarklet
- And many more professional mobile testing tools

All tools are **100% free to use** on the live site with no sign-up required.

## Repository Structure

This is a static GitHub Pages site built with HTML, CSS, and vanilla JavaScript.

Key files:
- `index.html` – Main landing page + core Mobile Viewer tool
- `device-frame-simulator.html`, `multi-device-preview.html`, `responsive-viewport-resizer.html`, etc. – Individual tool pages
- `styles.css` + `script.js` – Shared styles and functionality
- `header.js` / `footer.js` – Reusable components
- Blog & informational pages (SEO, responsive design guides, etc.)
- Legal pages (`privacy-policy.html`, `terms-of-service.html`)
